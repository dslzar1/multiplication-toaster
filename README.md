# Multiplication Toaster

Welcome to the "Multiplication Toaster" game! This is a fun math trivia game that challenges your multiplication skills against the clock. 

## Prerequisites

Make sure you have Python installed on your machine. You can download it from [python.org](https://www.python.org/downloads/).

## Running the Game

To start the game, run the following command in your terminal:

```bash
python3 src/game.py
```

Follow the on-screen instructions to enter your name and start playing. You will have 45 seconds to answer 5 multiplication questions. Your score will be displayed at the end, and you will have the option to try again.

## Enjoy the Game!

Challenge yourself and see how many questions you can answer correctly within the time limit!
